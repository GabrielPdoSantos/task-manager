# 📋 Sobre o Projeto

O TaskManager é um sistema de gerenciamento de tarefas desenvolvido em Java, executado via terminal (console). O projeto tem como objetivo aplicar conceitos fundamentais de programação e Programação Orientada a Objetos (POO).

---

## ⚙️ Funcionalidades

O sistema permite ao usuário:

* Criar novas tarefa
* Listar todas as tarefas cadastradas
* Marcar tarefas como concluídas
* Remover tarefas existentes

Toda a interação é feita por meio de um menu no console.


---

## 🖥️ Interface (Console)

O sistema funciona com um menu interativo:

===== GERENCIADOR DE TAREFAS =====
1. Criar nova tarefa
2. Listar tarefas
3. Marcar tarefa como concluída
4. Remover tarefa
5. Sair
Escolha uma opção:

O menu permanece em execução até o usuário escolher a opção de saída.

---

## 🧠 Conceitos Aplicados

### Fundamentos da Linguagem

- Entrada de dados com Scanner
- Saída de dados com System.out.println
- Estruturas de decisão (if, else, switch)
- Estruturas de repetição (for, while, do-while)
- Manipulação de listas com ArrayList
- Manipulação básica de String

## Programação Orientada a Objetos (POO)

- Encapsulamento com uso de private, getters e setters
- Herança (ex: TarefaPrioritaria)
- Polimorfismo no tratamento de tarefas
- Uso de classes e objetos
- Possível uso de classe abstrata ou interface
- Tratamento de exceções com try/catch
